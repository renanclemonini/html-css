Principais Formatações

Negrito / Destaque

->Nesta frase temos um termo em negrito usando a tag < b > (não semantica)

->Temos também um termo em negrito utilizando a tag <strong> (semantica)

Italico / Enfase

->Para escrever termo em italico usa-se a tag <i> (não semantica)

->Nesta frase temos um termo em ênfase utilizando a tag <em> (semantica)

Texto Marcado

->Podemos criar um texto marcado com a tag <MARK>

->Para alterar a cor do <mark> basta adicionar o parametro de css <mark style = "background-color: color;"> onde o mesmo ficará assim

Texto Grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse paragráfo.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido para indicar que ele foi adicionado depois.

Texto Sobrescrito e Subscrito

Textos sobrescritos são aqueles a exemplo do sinal de potenciação do numero x2 ao quadrado.

E textos subscrito são aqueles a exemplo de formúlas quimicas: H2O
